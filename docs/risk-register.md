# Risk Register — Mistral 7B RAG + Agent Pipeline

| ID | Category | Risk | Likelihood | Severity | Mitigation |
|---|---|---|---|---|---|
| R01 | Robustness | Mistral JSON parsing fails (observed 30% in M6) causing rule fallback and degraded task quality | High | Medium | Use structured output via Ollama format=json parameter or upgrade to mistral:14b |
| R02 | Robustness | Single T4 GPU: concurrent requests queue silently causing P99 > 30s | High | Medium | Add request queue with 10s timeout; alert at P99 > 20s in Prometheus |
| R03 | Robustness | Static 10-doc corpus: queries outside corpus scope trigger hallucination instead of abstention | High | High | Add retrieval confidence threshold: if top-1 sim < 0.70 return I don't know |
| R04 | Robustness | Ollama process crash causes full pipeline failure with no fallback | Medium | Critical | Systemd watchdog for Ollama; Prometheus alert at agent_ollama_model_loaded == 0 |
| R05 | Privacy | Query logs retained indefinitely on GCP VM disk in agent_traces/ directory | Medium | High | 30-day rolling deletion; avoid logging query content; log only hash and metadata |
| R06 | Privacy | Indirect prompt injection: malicious doc content in ChromaDB manipulates Mistral output | Medium | High | Sanitize retrieved chunks; add RETRIEVED_CONTENT delimiters in system prompt |
| R07 | Bias | 10-doc corpus over-represents ChromaDB and MLflow: biased tool recommendations | Medium | Medium | Expand corpus to 50+ docs with diverse tooling coverage before any production use |
| R08 | Compliance | No PII scanner on document ingestion pipeline | Low | Critical | Add Microsoft Presidio scan on all new documents before indexing into ChromaDB |
| R09 | Compliance | No AI output disclaimer: users may treat Mistral output as authoritative | Medium | High | Add disclaimer to all API responses: Answer generated by AI. Verify with primary sources. |
| R10 | Robustness | Context window overflow: 5 chunks x 512 tok + prompt approaches 8k Mistral limit | Low | Medium | Enforce max 4 chunks; add token budget check before generation call |
